# Restaurant Ordering App

A simple and interactive restaurant ordering application built with vanilla JavaScript, HTML, and CSS. Users can browse the menu, add items to their cart, manage quantities, and complete orders with a mock payment form.

## Features

- **Menu Browsing**: View available items with descriptions, ingredients, and prices
- **Cart Management**: Add items to cart, increase/decrease quantities, remove items
- **Order Summary**: Real-time total calculation and order overview
- **Payment Simulation**: Mock payment form with card details
- **Order Confirmation**: Success message after completing an order
- **Theme Toggle**: Switch between light and dark modes
- **Responsive Design**: Works on different screen sizes

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling with custom properties for theming
- **Vanilla JavaScript (ES6 Modules)**: Interactive functionality without frameworks

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd restaurant-ordering-app
   ```

2. Open `index.html` in your web browser.

No additional dependencies or build steps required - it's a pure client-side application!

## Usage

1. **Browse Menu**: View the available food and drink items
2. **Add to Cart**: Click the "+" button next to any item to add it to your order
3. **Manage Cart**: Use the "remove" button to decrease quantity or remove items entirely
4. **Complete Order**: Click "Complete Order" when ready
5. **Payment**: Fill in the mock payment form (name, card number, CVV)
6. **Confirmation**: Receive order confirmation message
